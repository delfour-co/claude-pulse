# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-15

### Added

- GNOME Shell panel indicator showing active Claude Code agent count
- Dropdown menu with agent details: type, project name, duration, profile
- Desktop notifications when agents finish
- Multi-profile support via `CLAUDE_PROFILE` environment variable
- Auto-detection of all `~/.claude*` configuration directories
- Worktree path stripping for clean project name display
- Install/uninstall/test scripts
- Claude Code hook script for lifecycle event capture
